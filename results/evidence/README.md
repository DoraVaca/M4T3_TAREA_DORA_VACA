# Evidencias de entrenamiento e inferencia

Esta carpeta contiene las **capturas y ejemplos visuales** que respaldan el desempeño del modelo YOLOv8.

## 📂 Contenido
- **Anotaciones (3–5 ejemplos)**  
  *Capturas de Roboflow mostrando cómo se etiquetaron las clases en el dataset.*

- **Predicciones en validación (10 ejemplos)**  
  *Imágenes del conjunto de validación con bounding boxes generados por el modelo.*

- **Predicciones en imágenes nuevas (5 ejemplos)**  
  *Evidencias de inferencia sobre imágenes distintas al dataset, para mostrar capacidad de generalización.*

- **Prueba de reproductividad**  
  *Evidencia de que el notebook se reproduce por otros usuarios.*
    

## 🎯 Uso
Estas evidencias permiten:
- **Verificar** la calidad de las anotaciones originales.  
- **Evaluar** el desempeño del modelo en datos conocidos (validación).  
- **Observar** cómo generaliza en datos nuevos.  

👉 Para un análisis detallado de errores, consultar el archivo **[`/docs/error_analysis.md`](../docs/error_analysis.md)**.
