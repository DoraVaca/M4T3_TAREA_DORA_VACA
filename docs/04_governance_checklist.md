# Checklist de Gobernanza

## Privacidad y consentimiento
- Los datos provienen de https://unsplash.com/es y de algunas se incluyen variaciones generadas utilizando RENDAIR.
- Se asegura consentimiento explícito o uso permitido por licencia.

## Minimización de datos
- Solo se incluyen imágenes necesarias para entrenar el modelo.
- Se evita recolectar información personal o sensible.

## Declaración de limitaciones
- El modelo **no debe usarse** en decisiones legales ni casos de seguridad crítica.
- Solo válido para fines académicos y experimentales.

## Nota de riesgos (FN vs FP)
- **Falsos positivos (FP):** pueden generar detecciones erróneas de objetos inexistentes.  
- **Falsos negativos (FN):** pueden omitir objetos relevantes, afectando la confiabilidad.  
- Se recomienda revisión humana en aplicaciones sensibles.

## Licencia
- Código: **Uso interno/privado**, sin autorización para redistribución ni modificación.  
- Dataset: Fotografías públicas de **Unsplash**, respetando sus términos de uso.  
- Restricciones: El código y el dataset se utilizan únicamente para fines académicos y de investigación personal.
