# 📂 Portafolio de Prototipo AECO CV — YOLOv8

Este repositorio reúne la documentación y análisis relacionados con la detección automática de **equipos de protección personal (EPP)** en el sector **AECO (Arquitectura, Ingeniería, Construcción y Operaciones)**.  
El objetivo es mostrar de manera transparente el planteamiento del problema, la definición de clases, los errores encontrados y las pautas de gobernanza para asegurar reproducibilidad y cumplimiento normativo.

## 📑 Contenido de la carpeta

1. **01_planteamiento_del_problema.md**  
   Explica el contexto del sector AECO, la importancia de la seguridad laboral y el objetivo del proyecto de detección automática de EPP.

2. **02_definiciones_de_clases.md**  
   Lista y describe las clases utilizadas en el dataset (ej. casco, chaleco, guantes, incumplimientos como *no_helmet* o *no_safety_vest*).

3. **03_análisis_de_errores.md**  
   Documenta los fallos del modelo (falsos positivos y negativos), con explicaciones sobre sus causas y posibles mejoras.

4. **04_governance_checklist.md**  
   Incluye una lista de verificación de gobernanza para garantizar transparencia, reproducibilidad y cumplimiento de buenas prácticas.



## 🎯 Objetivo del repositorio

- **Visibilizar éxitos y fallos** del modelo de detección.  
- **Facilitar auditorías** mediante documentación clara y estructurada.  
- **Promover la cultura de seguridad** en proyectos AECO.  
- **Asegurar reproducibilidad** con guías y checklist de gobernanza.  



## 🚀 Uso recomendado

Este portafolio puede servir como:
- Evidencia técnica e investigación académica
